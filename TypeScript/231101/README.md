# [DAY8] TypeScript - 1

## TypeScript Compiller 설치

```shell
sudo npm i -g typescript # 맥은 앞에 관리자권한 꼭 붙여주기!
```

## TypeScript Compiller 실행

```shell
tsc 파일명.확장자
```

## 타입스크립트 설정파일 생성

```shell
tsc --init
```

## watch로 ts 파일 변경 감지 + 자동 컴파일 진행

```shell
tsc --watch
```
